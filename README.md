# TRANSCEND Eligibility Checker

A clinical decision support tool for screening kidney transplant recipients for eligibility in the TRANSCEND Phase 3 trial (NCT06685757).

## About TRANSCEND

The TRANSCEND trial evaluates **Felzartamab** for treating late antibody-mediated rejection (AMR) in kidney transplant recipients.

## Disclaimer

This tool is a decision support aid and not a substitute for medical diagnosis or clinical judgment. Always refer to the full TRANSCEND protocol for complete eligibility criteria.

## Trial Information

- **Protocol**: NCT06685757
- **Phase**: 3
- **Intervention**: Felzartamab
